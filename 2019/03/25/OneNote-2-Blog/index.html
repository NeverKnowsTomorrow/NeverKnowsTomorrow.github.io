入坑计算机记

发表于 2019-03-25 更新于 2019-09-05 分类于 随笔 阅读次数： Valine：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

缘起

一切还要从 2013 年微软收购诺基亚说起。那时候出于对诺基亚的品牌信任，花了 2000 块入手了人生中第一台 Windows Phone - Lumia 900。殊不知，从此入坑好几年。WP 孱弱的软件生态，实在是和安卓、IOS 没法比啊。很多软件都没有 WP 客户端，主流的软件像 QQ、微信、支付宝，虽然有，但也是功能严重缺失，也就是勉强可以用的状态。其实这样也有好处，起码大大减少了玩手机的时间，让我可以专心学习（微笑脸）。

因为没有安卓苹果丰富的软件生态，只能在有限且质量普遍不高的软件里面尝试挑选精品软件，或者是挖掘系统自带的软件功能。在这个过程中，尝试了 OneNote、IT 之家、网易云音乐等优质的 WP 应用。这里要点名表扬一下 IT 之家，在当时 IT 之家属于 WP 生态里面品质最好的软件之一了。从那时起养成了每天在 IT 之家看科技新闻的习惯，直到现在换了小米 6，它也是我每天打开频率最高的软件之一。那时候 IT 之家是 WP 粉丝大本营，很多诺基亚粉丝、微软粉丝聚集在 IT 之家。评论区里有很多 WP 手机独有的紫色小尾巴，网友们也经常争论各大系统孰优孰劣，还有各种段子，没事看看评论区也挺有意思的哈哈。IT 之家的编辑们也大部分是微软的粉丝，每天都看到很多关于微软，关于 WP 的新闻。因为一直在用 WP，所以我也很关注 WP 的动态，盼着微软爸爸能更新系统，带领一众 WP 手机崛起。可是恨铁不成钢啊，WP7 系列的手机被微软放弃了，不能升级到 WP8，我的 Lumia 900 用了 2 年吧，充电口坏了，没法充电了，屏幕也碎了，只好换了新手机，系统停留在 WP 7.8。

买新手机时，我竟然已经是微软粉丝了，这挺奇怪的，被这残废的手机折腾了两年，竟然慢慢喜欢上了微软。我觉得很大一部分原因是 IT 之家看多了，天天关注 WP 新闻，关注微软的动态，正所谓日久生情吧。所以，新手机也是买的 WP，Lumia 638，骁龙 400 的低端机，WP8.1 的系统。但挺流畅的。后来微软推出了 win10 mobile 系统，以及伟大的 UWP 架构，我的 638 也可以升级。于是从预览版就第一时间升级体验了，比 WP8 还是慢不少的，没有 WP8 的丝般顺滑，当然我这手机配置也太烂了。升级了 win10 mobile 后可以安装 UWP 软件，我也是看着 UWP 软件生态一点点成长，体验过很多 UWP 软件，这里要点名表扬的是网易云音乐（虽然 UWP 客户端现在很久没更新了）和爱奇艺，这两个是优质 UWP 软件的代表。因为微软画的大饼，我还安利了我弟买了 Lumia 640，当初还希望 UWP 能一统安卓 IOS，可惜啊，微软在纳德拉的带领下，win10 mobile 还是没有成功崛起，后来微软放弃了 win10 mobile，裁撤了手机部门，宣告了微软在移动互联网领域的失败。我也没有必要再坚持了，换了小米 6，真香！

在这个过程中，虽然被微软始乱终弃折腾得不轻，但也让我逐渐喜欢上了微软的服务。win10 和 office 365、OneNote、VS code 等软件深入到了我的生活和学习中。我也是在这个过程中喜欢上科技，喜欢上计算机，折腾过各种软件，玩过各种 Linux 发行版，深深爱上了开放共享的开源世界。作为软粉，大学期间给十几个同学的笔记本升级了 win10，解决过各种电脑故障。在图书馆借来 Linux 相关的书自己研究，自学 Java 编程，这种喜欢越来越强烈，让我后悔为啥大学没选计算机专业。转专业是来不及了，于是我产生了跨考计算机研究生的想法，走上了一条艰难的考研之路。因为大学读的是电气工程与自动化专业，计算机相关的专业课只上过 C 语言和计算机网络，还都是比较水的课。我当时想着在准备考研的过程中补一补计算机的专业课，于是蜜汁自信的选择了考专业课 408，专业课考四门：数据结构、计算机组成原理、操作系统、计算机网络，号称考研最难的专业课。想想当初真是太天真了，太高估自己的实力，报了浙大的计算机，结果第一年连专业课都没复习完，一战失败。给家里人打了个电话，心有不甘，毅然决定二战，一志愿还是浙大。二战虽然复习完了，但还是折戟在专业课上，被 408 虐得惨不忍睹。只好选择调剂，但起码有研究生可以读了，还是自己喜欢的专业。因为当初决定跨考计算机，所以本科毕设自己选了大数据的题目，自己搭建了 Hadoop 集群，期间踩坑无数，也在折腾当中一点点进步，顺利完成了毕设答辩。

虽然跨入计算机这个行业的过程很艰辛，但我庆幸当初做出了正确的决定，做自己喜欢的事，不后悔。
