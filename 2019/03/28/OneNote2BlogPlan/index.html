OneNote 笔记搬运计划

发表于 2019-03-28 更新于 2019-09-05 分类于 OneNote 笔记搬运计划 阅读次数： Valine：
本文字数： 488 阅读时长 ≈ 1 分钟


我大概是 2013 年开始使用 OneNote 的。最初用过印象笔记，后来迁移到了 OneNote，至今快 6 年时间了，积累了不少笔记。一直都想搭一个博客，把笔记搬运到博客上，分享一下几年来积累的经验和知识，也可以在互联网上留下自己的痕迹。

19 年 1 月份，在学校通过 Hexo + GitHub pages 搭建了第一个个人博客，为主题效果折腾了挺长时间，后来删繁就简，保留了简约的风格，毕竟作为技术博客，重要的是内容。博客搭好以后，发布的第一篇博客是我翻译的 IPFS 白皮书，花了接近一周的时间。后来忙其他事又把博客暂时搁置了。写博客是件挺耗费时间和精力的事，尤其是高质量的博客，必定是精心雕琢打磨之后的作品。一直都特别欣赏坚持写高质量博客的人，好的博客可以把复杂的理论讲得清晰易懂，看起来也赏心悦目，更重要的是能对自己有所启发。想写一些高质量博客，但总不愿花太多时间精力，所以就一直拖延到现在。

最近看到一句话，深以为然。“人总是高估一天能做的事情，低估一年能做的事情”。所以，有意义的事情，坚持去做了，成果总会随时间积淀下来。开始写博客吧，先把 OneNote 笔记搬运计划执行起来，一年以后看看成果。
